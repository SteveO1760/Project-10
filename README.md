# BudgetBuddy Project 2

## How to run
1. Install Python 3
2. Open terminal in project folder
3. Install dependencies
4. Run the Project:

## Project Structure 
- Classes_10.py: Contains Budget class
- file_manager.py: Handles saving/loading expenses
- gui.py: Tkinter interface
- main.py: Launches the GUI
- expenses.txt: Stores saved expenses

## Features
- GUI for entering expenses
- File saving and loading
- Multiple categories supported
